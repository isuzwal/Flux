# Enops Landing Page

A modern, AI-powered landing page for database schema design, built with Next.js, Tailwind CSS, and smooth animations.

## ✨ Features

- **Beautiful Hero Section:** Eye-catching headline with animated text and a dynamic landing image.
- **Modern UI:** Built with Tailwind CSS for rapid, responsive design.
- **Smooth Animations:** Leveraging the `motion` library for engaging transitions and effects.
- **Feature Highlights:** Clear, icon-driven sections to showcase product strengths.
- **FAQ Accordion:** Interactive, accessible FAQ section.
- **Consistent Branding:** Custom logo, color palette, and typography.
- **Fully Responsive:** Looks great on all devices.

## 🖌️ Design & Animation

- **Tailwind CSS** is used for all styling, ensuring a clean and maintainable codebase.
- **Motion Animations:** Key elements (hero text, images) animate in on load for a polished feel.
- **Component-Driven:** All sections (navbar, hero, features, FAQ, footer) are modular React components.

## 🚀 Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```
2. **Run the development server:**
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the landing page.

## 📁 Structure

- `app/` — Main Next.js app entry and layout
- `components/` — UI components (hero, features, FAQ, footer, etc.)
- `public/images/` — Static assets (logo, landing image)

## 🛠️ Tech Stack

- **Next.js** — React framework for SSR and routing
- **Tailwind CSS** — Utility-first CSS framework
- **motion** — Animation library for React
- **Lucide Icons** — Modern icon set

## 🎨 Customization

- Update colors, fonts, and images in `globals.css` and `public/images/`.
- Edit or add sections in the `components/` directory.
